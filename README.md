# 🚀 Ouro do Dev — Manual Profissional Mateus Ernandes

O **Ouro do Dev** é um manual profissional de carreira front-end criado para servir como biblioteca de consulta, estudo, validação e evolução técnica de Mateus Ernandes.

O objetivo do projeto é reunir, em um único lugar, o padrão de trabalho para criar projetos front-end com organização, acessibilidade, responsividade, qualidade visual, boas práticas e preparação para portfólio.

## 🎯 Objetivo

Este manual existe para apoiar a construção de projetos front-end com método profissional.

A ideia central é:

> Eu não preciso decorar tudo. Eu preciso dominar o processo, consultar meu padrão, aplicar com qualidade, testar com ferramentas e evoluir com projetos reais.

## 🎨 CSS ME Standard

O **CSS ME Standard** é o padrão pessoal de organização CSS de Mateus Ernandes.

Ele segue a ideia:

1. Primeiro eu estruturo.
2. Depois eu nomeio.
3. Depois eu estilizo.
4. Depois eu refino.
5. Depois eu valido.
6. Depois eu documento.

## 🔍 Ferramentas recomendadas

Todo projeto deve ser validado com:

- DevTools do navegador
- Lighthouse
- W3C HTML Validator
- W3C CSS Validator
- axe DevTools
- HTMLHint
- Teste visual no celular e desktop
